Software

Software (GUI)

Overview

The software section of our project included the user facing GUI. We decided to use Kivy, a python library to create the GUI. The user would plug their computer into the system and the GUI would tell them the status of each of their four plants.

Code

You can view our code here: https://github.com/aditisjoshi/dankherb. Our github includes both the python and arduino code as well as the code for the website.

Interfacing with Arduino

The python script interfaces with our arduino code and to collect data from the arduino and create a GUI that is a popup on someoneâ€™s computer. The data is collected in real time from the two sensors for each of the four plants (eight in total). The script then creates an average of all of the sensor values and checks them against each of the type of plant that they are for. It reports to the user the status of each of the plants so they can take any necessary action.

The Application

The python script includes two different classes: a plant class and a garden class. When initializing the application the user can input the number, names, and types of their plants when making a garden instance and the garden makes the number and types of plants that the user inputted. Then the user can run the app and find the necessary information about each of their plants.
